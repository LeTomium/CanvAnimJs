About

CanvAnim is a front-end library that allows you to improve your HTML canvas by making your items much easier to handle. That use Oriented Object.

The main purpose of Canvanim is to simplify the handling of items in your canvas.

Example

                
                    const el = document.querySelector("canvas.canvanim")
                    const area = new CanvAnim(el, {
                        scrollRegion: [50, 50, 50, 50]
                    })
        
                    let rect = area.createRectangle(50, 50, 100, 150, {
                        backgroundColor: "red",
                    borderWidth: 2
                    })
                    rect.on("click", (e) => {
                        console.log(`Rectangle in ${e.x}:${e.y}`)
                    })

Why?

  • Easy to use
  • Object-oriented
  • Speedy and efficient
